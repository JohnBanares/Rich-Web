Extension functionalities:
- change the background color of current page viewed
- changes the images of a page to random images of web technologies
- set the same inner text to all <h> tags and change their colors, font styles and add border. 
  This is done the same for all <p> tags.
- change the innertext of all <span> and <a> tags not relating to web using keywords
